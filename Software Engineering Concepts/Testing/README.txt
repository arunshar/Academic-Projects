Project consist of 4 java files
1) Members.java
2) Scores.java
3) ScoresGUI.java
4) ScoreGUITest.java

In order to run the application do the following :

1) Go to SEC_code/code/src/
2) Run ScoresGUI.java

In order to test the application do the following :

1) Go to SEC_code/code/src/
2) Run ScoresGUITest.java
